# titanic-analysis

## Objective
The goal of this project is to explore the Titanic dataset and understand what factors affected passenger survival.

## Dataset
The dataset is taken from Seaborn and includes information such as age, sex, class, and survival.

## Analysis
- Explored the dataset using basic functions (head, info, describe)
- Cleaned missing values in the age column
- Removed the deck column due to many missing values
- Used groupby to analyze survival by sex, class, and age

## Visualisations
- Survival count
- Survival by gender
- Survival by class
- Age vs survival
- Relationship between variables using line plots

## Key Findings
- Females had a higher survival rate than males
- First-class passengers had better survival chances
- Age had some influence on survival
- Survival was affected by multiple factors

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
